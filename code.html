The Code

This program was written in Python using the cv2 and numpy packages. By importing a pre-trained neural network, we are able to effectively colorize black and white images. The credits for the neural network go to Richard Zhang, a research scientist at Adobe who has a PhD in EECS from UC Berkely. The credits for the script that runs the program go to OpenCV on Github. I recreated the program in my own IDE and ran the script on various photos that are displayed on the main page.
